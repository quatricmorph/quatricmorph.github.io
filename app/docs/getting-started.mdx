# Getting Started

This guide will help you get started with ViệtLLM quickly.

## Installation

```bash
npm install quatricmorph
```

## Quick Example

```javascript
import { Quatricmorph } from 'quatricmorph'

const model = new Quatricmorph({
  apiKey: 'your-api-key'
})

const response = await model.generate('Xin chào!')
console.log(response)
```
