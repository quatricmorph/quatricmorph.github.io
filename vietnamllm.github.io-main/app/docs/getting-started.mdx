# Getting Started

This guide will help you get started with ViệtLLM quickly.

## Installation

```bash
npm install vietnamllm
```

## Quick Example

```javascript
import { VietnamLLM } from 'vietnamllm'

const model = new VietnamLLM({
  apiKey: 'your-api-key'
})

const response = await model.generate('Xin chào!')
console.log(response)
```
