# About ViệtLLM

ViệtLLM is a community-driven initiative focused on advancing Large Language Model research and development in Vietnam.

## Our Mission

We aim to:

- Foster AI research and development in Vietnam
- Build open-source LLM tools and resources
- Create educational content about AI and machine learning
- Connect researchers, developers, and enthusiasts
